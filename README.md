# Student Portal System

## Overview
This project is a **Student Portal System** login and registration UI, designed for web applications. It demonstrates a clean and responsive user interface with modern form functionality, including **show/hide password**.

The project was built as part of **Laboratory 2: Advanced UI Implementation** for learning **HTML, CSS, and JavaScript** with a focus on accessibility and user experience.

## Features

- **Login Form** with email and password
- **Register Form** with full name, email, password, and confirm password
- **Show/Hide Password** toggle for better UX
- **Responsive UI** for desktop and mobile
- **Single-page form switch** between Login and Register
- Clean card layout with modern design

## Technologies Used

- **HTML**
- **CSS**
- **JavaScript**
- **Font Awesome** for icons
